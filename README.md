# Mixing Board

Interactive energy allocation tool from Veris Coaching.

Live: https://scottmcbean.github.io/mixing-board/

A prototype for exploring how people distribute psychological energy across nine patterns — and what happens when the mix shifts. Single-file HTML, no build step. Update by editing `index.html` and pushing to `main`.
